# Cricket Scoring App

A modern, responsive cricket scoring web application built with React, Vite, and TailwindCSS.

## Features

- Express and Custom scoring modes
- PWA support for offline functionality
- Modern, responsive UI inspired by CRED design
- Real-time scoring and statistics
- Team and player management
- Comprehensive match history

## Tech Stack

- React + TypeScript
- Vite
- TailwindCSS
- Shadcn/ui
- Storybook
- Vitest + Testing Library
- Playwright
- PWA Support

## Getting Started

### Prerequisites

- Node.js 18+
- pnpm 8+

### Installation

```bash
# Clone the repository
git clone https://github.com/yourusername/react-cricket-scoring-app.git

# Navigate to the project directory
cd react-cricket-scoring-app

# Install dependencies
pnpm install
```

### Development

```bash
# Start development server
pnpm dev

# Run Storybook
pnpm storybook

# Run tests
pnpm test

# Run E2E tests
pnpm e2e

# Build for production
pnpm build
```
